##Отчеты по практическим работам

- [Практическая работа №1](https://github.com/user-attachments/files/25675941/1.docx) Реализовал карточку товара (содержимое: название, описание,
фотография) с применением переменных (минимум 2), миксина (минимум 1) и
использованием вложенной структуры селекторов на любом из двух препроцессоров
(SASS или LESS).

- [Практическая работа №2](https://github.com/user-attachments/files/25675955/2.docx) Реализовал API, которое предоставляет CRUD операции для списка
товаров (просмотр всех товаров, просмотр товара по id, добавление товара,
редактирование товара, удаление товара). Объект товара должен содержать
следующие поля: id, название, стоимость.

- [Практическая работа №4](https://github.com/user-attachments/files/25675968/4.docx) Необходимо доработать рассмотренный пример: два небольших приложение на React и
Express (для клиента и сервера, соответственно), связанных в единую систему - сайт
интернет-магазина товаров с тематикой на ваше усмотрение. Количество товаров - не
менее 10. Карточка товара должна содержать как минимум: название, категорию,
описание товара, цену, количество на складе. Опционально доработать рейтинг и фото.

- [Практическая работа №5](https://github.com/user-attachments/files/25675970/5.docx) Подключил swagger-jsdoc и swagger-ui-express , описал с помощью JSDoc-аннотаций схему пользователя ( User ) и все CRUD-операции ( GET , POST , GET/:id , PATCH/:id , DELETE ). Убедился, что документация доступна по адресу /api-docs и работает в интерактивном режиме (можно отправить тестовый запрос). 

### Бэкенд
- **Node.js** — среда выполнения JavaScript
- **Express.js** — веб-фреймворк для создания REST API
- **nanoid** — генерация уникальных идентификаторов
- **cors** — middleware для обработки CORS-запросов
- **swagger-jsdoc** — генерация OpenAPI спецификации из JSDoc-комментариев
- **swagger-ui-express** — отображение интерактивной документации Swagger

### Фронтенд
- **React.js** — библиотека для построения пользовательского интерфейса
- **Axios** — HTTP-клиент для запросов к API
- **Sass (SCSS)** — препроцессор CSS для стилизации компонентов
- **React Scripts** — сборка и запуск проекта
